What is a Zine?

A zine is a self published magazine. I've heard that in the times before github people used to share code via zine.

Why did you create this zine?

I created this zine for the Tech Lady Hackathon in 2015 to complement other classes that were being run. I thought it would be really neat to teach code with a zine.
